## 一、使用说明

本文件用于维护具体天赋条目。所有条目先进入“候选”，经过讨论后再调整为“已确认”“已实现”“待测试”“调整中”或“废弃”。

本文件记录天赋本身，不记录阶段任务、实现验收或路线图状态。阶段实现范围进入 [[策划案_天赋系统]]，任务状态进入 [[INDEX_开发路线图]]。

## 二、状态说明

| 状态 | 含义 |
|------|------|
| 候选 | 想法可用，但尚未确认进入正式池 |
| 已确认 | 设计上成立，等待实现或测试 |
| 已实现 | 已进入 Godot 数据或代码 |
| 待测试 | 已实现但需要 playtest 验证 |
| 调整中 | 体验、数值或描述需要修改 |
| 废弃 | 不再使用，保留原因便于回溯 |

## 三、关键词 Tooltip 词典

| 关键词 | Tooltip |
|--------|---------|
| 共振 | 某个动作成功后，强化或改变下一次指定动作。 |
| 回流 | 能量被消耗后，满足条件时返还部分能量。 |
| 裂纹 | 附着在敌人身上的不稳定痕迹。达到指定层数时触发额外效果。 |
| 过载 | 短时间获得强化，同时承担额外代价或限制。 |
| 护势 | 由动作产生的短暂保护，可抵消伤害、减少失败惩罚或保留关键状态。 |

## 五、候选天赋条目

### 偏斜共振

| 字段 | 内容 |
|------|------|
| 天赋 ID | `talent_deflect_resonance` |
| 层级 | 二级 |
| 主流派 | 弹反 |
| 副流派 | 普攻 |
| 关键词 | 共振 |
| 机制类型 | 桥梁 |
| 关联动作 | 弹反、普攻、超频普攻 |
| 触发条件 | 弹反成功后 |
| 效果描述 | 弹反成功后，获得 1 次共振。下一次普攻视为超频普攻。 |
| Tooltip | 共振：某个动作成功后，强化或改变下一次指定动作。 |
| 代价 | 仍按超频普攻规则消耗能量；能量不足时只触发普通普攻。 |
| 组合对象 | 超频普攻消耗降低、超频击杀回流、普攻裂纹相关天赋 |
| 设计意图 | 把弹反成功转化为主动进攻窗口，让弹反流自然接入普攻和超频路线。 |
| 风险点 | 如果超频普攻收益过高，可能让弹反流只追求这一个桥梁。 |
| 当前状态 | 候选 |
| 备注 | 需要确认“视为超频普攻”是否要求玩家额外按住超频键。 |

### 过载回流

| 字段 | 内容 |
|------|------|
| 天赋 ID | `talent_overclock_backflow` |
| 层级 | 二级 |
| 主流派 | 超频 |
| 副流派 | 能量 |
| 关键词 | 回流 |
| 机制类型 | 稳定性 / 消耗 |
| 关联动作 | 超频动作 |
| 触发条件 | 超频动作击杀敌人时 |
| 效果描述 | 超频动作击杀敌人时，返还本次超频消耗的 50% 能量。 |
| Tooltip | 回流：能量被消耗后，满足条件时返还部分能量。 |
| 代价 | 只有击杀触发；未击杀不返还。 |
| 组合对象 | 超频普攻、超频闪避伤害化、过载类短时爆发天赋 |
| 设计意图 | 降低超频流的净消耗，让高频超频有自循环可能。 |
| 风险点 | 面对低血量敌群时可能过强，需要结合敌人密度测试。 |
| 当前状态 | 候选 |
| 备注 | 返还比例可从 30% / 50% / 100% 三档测试。 |

### 裂纹斩

| 字段 | 内容 |
|------|------|
| 天赋 ID | `talent_crack_slash` |
| 层级 | 一级 |
| 主流派 | 普攻 |
| 副流派 |  |
| 关键词 | 裂纹 |
| 机制类型 | 入口 / 引擎 |
| 关联动作 | 普攻 |
| 触发条件 | 普攻命中敌人时 |
| 效果描述 | 普攻命中敌人时，施加 1 层裂纹。裂纹达到 5 层时爆发，造成额外伤害并清除层数。 |
| Tooltip | 裂纹：附着在敌人身上的不稳定痕迹。达到指定层数时触发额外效果。 |
| 代价 | 需要连续命中同一目标，切目标会降低成型速度。 |
| 组合对象 | 普攻攻速、弹反后普攻强化、裂纹爆发增伤类天赋 |
| 设计意图 | 给普攻流一个可读的成长标记，让普攻从稳定输出走向爆发循环。 |
| 风险点 | 层数过高会拖慢反馈，层数过低会变成无脑爆发。 |
| 当前状态 | 候选 |
| 备注 | 需要决定裂纹是否有持续时间，是否允许多个敌人分别记录层数。 |

### 逆流过载

| 字段 | 内容 |
|------|------|
| 天赋 ID | `talent_reverse_overload` |
| 层级 | 三级 |
| 主流派 | 超频 |
| 副流派 | 能量 |
| 关键词 | 过载 / 回流 |
| 机制类型 | 核心 |
| 关联动作 | 超频动作、能量获取 |
| 触发条件 | 连续触发 3 次超频动作后 |
| 效果描述 | 进入 4 秒过载。过载期间超频动作消耗降低 50%，超频击杀触发回流。过载结束后 3 秒内无法通过弹反获得能量。 |
| Tooltip | 过载：短时间获得强化，同时承担额外代价或限制。回流：能量被消耗后，满足条件时返还部分能量。 |
| 代价 | 过载结束后的回能封锁会制造资源真空。 |
| 组合对象 | 超频消耗降低、超频击杀、敌群清理、护势类失败保护 |
| 设计意图 | 定义超频流的核心循环：连续投入能量，换取短时间压制，再承担资源低谷。 |
| 风险点 | 如果清怪速度过快，回能封锁可能没有实际惩罚；如果封锁过重，玩家会不敢触发。 |
| 当前状态 | 候选 |
| 备注 | 三级候选，适合后续完整天赋池讨论。 |

| 主流派 | 弹反 |

| 机制类型 | 稳定性 |
| 关联动作 | 弹反、受伤 |

### 余热回流

| 字段 | 内容 |
|------|------|
| 天赋 ID | `talent_residual_backflow` |
| 层级 | 一级 |
| 主流派 | 超频 |
| 副流派 | 能量 |
| 关键词 | 回流 |
| 机制类型 | 稳定性 / 消耗 |
| 关联动作 | 超频普攻 |
| 触发条件 | 超频普攻命中敌人时 |
| 效果描述 | 超频普攻命中敌人时，返还 1 点能量。 |
| Tooltip | 回流：能量被消耗后，满足条件时返还部分能量。 |
| 代价 | 空挥不返还；需要先支付超频普攻消耗。 |
| 组合对象 | 超频普攻消耗降低、偏斜共振、超频击杀回流 |
| 设计意图 | 给超频路线一个轻量稳定性入口，让玩家感受到“花能量强化动作后，可以通过命中回收一部分成本”。 |
| 风险点 | 如果返还值过高，会让超频普攻失去消耗感；如果过低，玩家可能感知不到。 |
| 当前状态 | 候选 |
| 备注 | 适合作为早期三选一验证候选，效果简单且能直接检验回流关键词。 |
