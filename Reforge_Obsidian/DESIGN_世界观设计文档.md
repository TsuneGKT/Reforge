# Reforge（重锻）· 世界观设计文档

> **类型**：2D俯视角动作Roguelike
> **美术风格**：像素风格
> **世界观基调**：中世纪衰败感，十字军东征为历史参考
> **叙事重量**：中等——有一条暗线但不强制玩家关注，藏在环境和Boss对话中
> **核心叙事主题**：保护者变成被审判者——你以为自己在行使正义，直到发现正义本身是谎言

---

## 一、世界设定

### 1.1 王国与骑士团

大陆的边境之外存在一种来源不明的力量——**异力**。异力能改变一切与它接触的事物：动物变异、植物腐化、金属锈蚀、地形扭曲。

为了抵御异力的侵蚀，王国组建了**边境骑士团**，在大陆边缘修筑堡垒、箭塔、防线，驻守了几十年。骑士团是王国最受尊敬的存在——他们守护着文明与异域之间的最后一道线。

### 1.2 异化与审判

但边境的战斗有一个所有人都不愿面对的事实：长期接触异力的骑士，身体会发生变化。最初是微小的——反应更快，力量更大，伤口愈合更快。王国睁一只眼闭一只眼，因为需要他们。

但变化会累积。到了某个程度，骑士的身体开始呈现明显的非人特征。

王国的选择是：**宣布骑士团为异端，发动审判，将他们全部处决或驱逐到边境之外。**

骑士团在一夜之间从守护者变成了被猎杀的对象。他们被赶入了自己曾经守护王国抵御的那片异域。王国封闭了边境，在身后筑起了高墙。

### 1.3 废墟现状

现在，边境外是一片彻底废墟化的世界——骑士团曾经的堡垒、前哨站、巡逻路线、训练场、教堂，混杂着异力侵蚀后扭曲的荒野和林地。

被驱逐的骑士们散落其中。有些还保持着理智，有些已经被异力彻底改变了形态。堡垒里的军犬、防御工事、生活设施也被异力渗透，变成了各种危险的存在。

只剩遗迹和残骸。没有秩序，没有文明，只有还在运转的旧机关和还在游荡的旧亡魂。

## 二、主角

### 2.1 身份

主角是王国派出的**审判执行者**。

大审判之后，王国需要确保被驱逐的骑士团被彻底清剿。于是培养了一批"审判者"——经过特殊处理的战士，被告知自己拥有"神赐的净化能力"，使命是深入边境废墟，消灭异端余孽。

主角完全相信官方叙事：骑士团背叛了王国，堕入异端，必须被消灭。主角带着使命感出发，认为自己在做正义的事。

### 2.2 主角不知道的事

主角之所以能吸收异力（弹反获取能量），不是什么"神赐能力"——是王国在派出审判者之前对他们的身体做了改造，让他们能兼容异力。

审判者不是正义的执行者，是王国的**收割工具**——派出去回收骑士团身上积累了几十年的异力，带回来给王国使用。

主角对此一无所知。直到在废墟中遇到那些还能说话的Boss。

### 2.3 主角的认知弧线

| 阶段 | 主角认知 | 对应游戏阶段 |
|------|---------|------------|
| 前期 | "我是正义的审判者，清除异端" | 第1-10房间，打铁期 |
| 中期 | "有些事好像不对劲……" | 第11-20房间，Boss对话开始动摇认知 |
| 后期 | "我可能是被利用的工具" | 第21-30+房间，自身异化已与骑士无异 |

## 三、核心机制与世界观对应

### 3.1 双层叙事表

每一个机制都有"主角以为的叙事"和"实际真相"两层含义。前期玩家只接触到第一层，随着推进逐渐看到第二层。

| 机制 | 主角以为 | 实际真相 |
|------|---------|---------|
| 弹反 | 神赐的净化能力，承接异端攻击时将其净化 | 王国改造过的身体才能做到，和骑士团的技术本质相同 |
| 能量 | 净化异端后回收的"圣能" | 从骑士残骸上掠夺的异力积累 |
| 超频 | 圣战姿态，释放净化之力 | 和骑士团在边境用的"异力共鸣"一模一样，只是被王国换了个名字 |
| 光核结晶 | 净化结晶 | 骑士团发明的异力稳定技术 |
| 天赋构筑 | 在净化祭坛上深化神恩 | 异力改造身体——和骑士团当年的异化是同一个过程 |
| 三级天赋 | 成为神之器皿 | 你的身体已经是"异端"了 |
| 技能（三级赋予） | 神恩的最终馈赠 | 异力具现化，完全超出人类范畴 |
| 每局重开 | 使命传承，新的审判者继续前人的任务 | 王国有的是消耗品可以替换 |
| Meta成长 | 前任审判者留下的经验继承 | 王国从前任审判者身上提取数据，优化了下一批的身体改造 |
| 祭坛 | 净化祭坛 | 骑士团的锻造台 |
| 结晶点 | 净化回收装置 | 骑士团的异力稳定装置 |
| 商店 | 王国的补给站 | 边境残余物资，或还保持理智的骑士在交易 |

### 3.2 双层命名切换系统

游戏内UI上的机制名称会随叙事推进发生切换。前期使用主角视角的命名，第二个Boss战后（主角认知开始动摇），UI名称自动替换为真相命名。玩家不会收到任何提示，自己发现"名字变了"本身就是叙事体验的一部分。

| 机制 | 前期名称（主角视角） | 后期名称（真相） | 切换时机 |
|------|-------------------|----------------|---------|
| 能量 | 圣能 | 异力 | 第二个Boss战后 |
| 超频 | 神恩释放 | 异力共鸣 | 第二个Boss战后 |
| 光核结晶 | 净化结晶 | 异力凝核 | 第二个Boss战后 |
| 天赋 | 神恩 | 异铸 | 第二个Boss战后 |
| 祭坛 | 净化祭坛 | 锻造台 | 第二个Boss战后 |
| 结晶点 | 净化回收点 | 异力稳定点 | 第二个Boss战后 |

**教程文本同步切换**：前期教程写"积攒圣能"，切换后自动变为"积攒异力"。如果玩家注意到教程也变了，这是额外的叙事细节——"连教程都在骗我"。实现成本低，只需两套文本字符串替换。

### 3.3 天赋构筑与异化的同步

天赋系统的三层递进在叙事上对应主角身体异化的三个阶段：

**一级天赋（强化层）**：身体发生微小变化——反应更快、力量更大、感官更敏锐。和骑士团早期异化完全一样。主角不会起疑，因为王国告诉过他"审判者会因神恩而变强"。

**二级天赋（改造层）**：身体开始出现明显的非人能力——闪避时身体短暂虚化，普攻附带冲击波，弹反后释放能量爆发。主角开始隐约觉得不对劲。这恰好也是中期Boss开始和主角对话、揭露部分真相的时候。

**三级天赋（核心层）**：彻底的形态改变。玩法质变，割草感来了。但主角此时已经和Boss们没什么区别——如果王国看到他现在的样子，他就是下一个被审判的对象。

### 3.4 五条流派的叙事色彩

| 流派 | 叙事含义 |
|------|---------|
| 弹反流 | 骑士团的核心战技发挥到极致——你用来杀他们的技术就是他们发明的 |
| 普攻流 | 异力强化后的纯粹物理输出，最接近骑士团全盛时期的战斗风格 |
| 闪避流 | 异力改变了身体的物理性质，能短暂虚化、瞬移——明显的非人能力 |
| 技能流 | 异力具现化为外放攻击，完全超出人类范畴 |
| 超频流 | 极致的异力共鸣，最接近Boss们失控前的状态 |

流派越强，主角看起来越不像人。玩家的爽感本身就是叙事的一部分——你越强，你就越像你来讨伐的"异端"。

## 四、区域设定

废墟分为三大区域，对应骑士团驻地从外到内的结构。越靠外异力浓度越高、异化越严重，越靠内越保留骑士团生活的痕迹。

### 4.1 荒野（边境外围）

骑士团日常巡逻和作战的林地与荒原。异力浓度最高，动植物异化最严重。地形扭曲，树木金属化，水源被污染。

视觉基调：锈绿与暗褐色调，扭曲的树木，被异力侵蚀的岩石和土壤。

### 4.2 防御工事（堡垒外层）

骑士团修建的军事设施——城墙、箭塔、城门、巡逻道。被驱逐后无人维护，异力渗透进了建筑和机关。石墙开裂，铁门锈蚀，但自动防御设施还在运转。

视觉基调：灰石与锈铁色调，半坍塌的城墙，还在机械运作的弩塔和陷阱。

### 4.3 骑士团内部（堡垒内层）

营房、训练场、教堂、锻造间。异化程度较轻，残留更多骑士团生活的痕迹——散落的私人物品、墙上的涂鸦和壁画、教堂里还在燃烧的异化烛火。叙事线索集中在这个区域。

视觉基调：暗色石质室内，微弱的异化光源，残破但能辨认出曾经用途的生活设施。

## 五、怪物设计

怪物按区域分类，每个怪物形态简单、行为清晰、美术成本可控。小怪就是练手的，前期单个出现，后期组合施压。

### 5.1 荒野怪物

#### 锈犬

骑士团边境军犬。活着时跟随骑士巡逻林地，死后被异力驱动。犬甲与躯体融合，关节锈蚀但行动依然敏捷。还保留着生前的巡逻本能——在固定路线上游荡，发现活物就扑咬。

| 属性 | 数值 |
|------|------|
| 生命值 | 40 |
| 攻击力 | 8 |
| 攻击频率 | 每2秒 |
| 移速 | 中 |

行为：朝玩家跑过来，到攻击距离后铁甲扑击。前摇明确（身体后蹲蓄力），弹反窗口大（16帧），弹反成功后弹开硬直1秒。前期单只出现，是玩家接触的第一个弹反教学对象。后期成群出现（3-5只）时，多只交错扑击打乱弹反节奏，逼迫在弹反和闪避之间切换。

#### 腐鹿

林地里的野鹿，长期饮用被异力污染的水源后异化。鹿角变成锈铁色尖刺，体型比正常鹿大一圈。性情从温顺变为暴躁。

| 属性 | 数值 |
|------|------|
| 生命值 | 80 |
| 攻击力 | 18 |
| 攻击频率 | 每3秒 |
| 移速 | 中 |

行为：低头冲撞。前摇明显（低头蓄力1秒），弹反窗口大（16帧），前期弹反教学怪。撞到墙会自己晕1秒。

#### 孢菇

林地地面长出的异化蘑菇。不会动，不会攻击，但玩家靠近时自动释放一团孢子云。

| 属性 | 数值 |
|------|------|
| 生命值 | 15 |
| 攻击力 | 3/秒（孢子云） |
| 移速 | 不动 |

行为：站在那里，靠近就放毒。一刀就能砍掉。存在意义是地形障碍——和其他怪配合时限制走位空间。

### 5.2 防御工事怪物

#### 残弩

城墙上的自动弩，异力渗透后恢复运转，无差别射击一切移动目标。曾经是骑士团用来保护边境的防御武器。

| 属性 | 数值 |
|------|------|
| 生命值 | 50 |
| 攻击力 | 30 |
| 攻击频率 | 每4秒 |
| 移速 | 不动 |

行为：蓄力后射一发。可弹反但窗口紧（8帧），弹反成功打回去双倍伤害。美术上是一个固定弩台，只有蓄力和射击两个状态。

#### 铁罐

骑士团设置的防御陷阱——装满碎铁片的铁罐，原本用绊线触发。异力渗透后感知到附近有活物就自动引爆。

| 属性 | 数值 |
|------|------|
| 生命值 | 1 |
| 攻击力 | 20（范围） |
| 移速 | 不动 |

行为：玩家接近到一定范围，发出1秒滴滴声后爆炸。可以提前砍掉，也可以听到声音后闪避。地形陷阱，逼玩家注意脚下。

#### 巡墙甲

城墙上的巡逻甲胄。骑士团用异力驱动的空甲，自动巡逻。骑士团被驱逐后这些空甲还在走，行为已经错乱，见到任何东西都视为入侵者。

| 属性 | 数值 |
|------|------|
| 生命值 | 100 |
| 攻击力 | 15 |
| 攻击频率 | 每2.5秒 |
| 移速 | 慢 |

行为：举武器劈一刀。前摇清晰，弹反窗口大（16帧），节奏稳定可预测。像素下就是一套走路的空盔甲。

### 5.3 骑士团内部怪物

#### 游魂灯

骑士团在堡垒内设置的长明灯，用来照明和标记安全区域。异力渗透后灯火脱离灯架，变成飘荡的小光团。不是有意攻击，只是不稳定，碰到就灼伤。

| 属性 | 数值 |
|------|------|
| 生命值 | 10 |
| 攻击力 | 5 |
| 移速 | 中，无规律飘荡 |

行为：房间里随机飘动，碰到玩家就造成伤害。一碰就灭。数量多（5-8个），制造走位干扰。曾经是骑士们在黑暗中标记"这里安全"的信号。

#### 甲鼠

堡垒里的老鼠，啃食了异力污染的军粮后变异。体型从拳头大变成小狗大，背上长出类似甲片的硬壳。

| 属性 | 数值 |
|------|------|
| 生命值 | 25 |
| 攻击力 | 6 |
| 攻击频率 | 每0.8秒 |
| 移速 | 快 |

行为：跑过来啃你。比锈犬更弱但更快，3-6只一起出现。锈犬的室内版本。

#### 残烛祭像

教堂里的圣像，骑士团信仰的象征，面前常年点着蜡烛。异力渗透后圣像裂开，烛火变成不灭的异火。

| 属性 | 数值 |
|------|------|
| 生命值 | 80 |
| 攻击力 | 4/秒（灼热光环） |
| 移速 | 不动 |

行为：站在那里烧。靠近就掉血，砍碎就灭了。区域限制——教堂房间里摆几个，安全走位空间就被切割了。

### 5.4 精英怪：异铸骑士

骑士不是普通小怪，是精英级别。每个异铸骑士都曾是骑士团成员，被驱逐后异化程度不同，但还残留着战斗本能。出场频率低，每次遇到都是一场小型考验。

异铸骑士的异化方向不同，对应不同战斗风格（剑、盾、弓等）。部分异铸骑士残留微弱的人性痕迹——击杀后单膝跪地行骑士礼再消散，或战斗中偶尔犹豫停顿。不解释，不弹字幕，让玩家自己感受。

（精英怪具体数值与行为待后续细化。）

### 5.5 怪物总览

| 区域 | 怪物 | 类型 | 核心行为 |
|------|------|------|---------|
| 荒野 | 锈犬 | 近战追击 | 铁甲扑击，可弹反 |
| 荒野 | 腐鹿 | 节奏冲撞 | 低头冲，可弹反 |
| 荒野 | 孢菇 | 地形障碍 | 靠近就放毒 |
| 工事 | 残弩 | 固定远程 | 蓄力射箭，可弹反 |
| 工事 | 铁罐 | 陷阱 | 接近后延迟爆炸 |
| 工事 | 巡墙甲 | 节奏近战 | 举武器劈，可弹反 |
| 内部 | 游魂灯 | 飘动干扰 | 随机飘，碰到就伤 |
| 内部 | 甲鼠 | 快速群体 | 室内版锈犬 |
| 内部 | 残烛祭像 | 区域限制 | 固定灼热光环 |
| 全区域 | 异铸骑士 | 精英 | 待细化 |

### 5.6 怪物组合设计原则

- 前期（第1-10房间）：单一类型怪物，单个或少量出现，玩家可用单一策略应对
- 中期（第11-20房间）：两类怪物混搭（如锈犬+孢菇、巡墙甲+游魂灯），逼迫动作切换
- 后期（第21-30+房间）：多类同时出现+精英怪，但此时Build成型，形成割草爽感

## 六、Boss设计方向

### 6.1 Boss的核心定位

Boss不只是战斗检验，更是**叙事推动器**。每个Boss是一个还清醒的被驱逐骑士，能对话，有完整的人格和立场。他们知道主角是王国派来的审判者，但反应各不相同。

主角前期认为Boss是"异端头目"，实际上他们是知道真相的人。每打一个Boss，主角（和玩家）的认知就被动摇一次。

### 6.2 Boss叙事递进

**第一个Boss：已经接受命运的老骑士**

异化程度较轻，还基本保持人形。他不想打，但主角不会听他说话——这时主角还完全相信"异端必须清除"。战斗中他会说几句话，比如"你身上的力量……和我们的一样"，但主角只会当成异端的蛊惑。

他的作用是**埋下第一颗种子**。玩家可能会想"等等他刚才说了什么"，但不确定。

**第二个Boss：愤怒的年轻骑士**

异化程度中等，身体已经开始扭曲，但战斗风格还能看出受过正规训练。她对王国充满愤怒，战斗中喊出更具体的信息碎片——"我们为他们守了三十年边境"、"异力是从王都地下渗出来的"。

主角此时可能开始觉得不对劲，但还没有动摇。动摇的是玩家。

**第三个Boss（最终Boss前）：骑士团团长**

异化程度很深，体型已远超人类，但意识完全清醒。他是唯一一个先和主角对话再开打的Boss。他把真相的关键部分说出来：骑士团发现了异力的真正来源在王都内部，试图向王国揭露，结果被灭口。

关键台词——"你以为你能用我们的力量是因为你是审判者？你只是下一批实验品。"

他可能已经异化到无法控制身体的程度——意识清醒但身体不听使唤。他一边和主角打，一边在道歉。

### 6.3 Boss与流派克制

Boss的阶段切换考验Build的多元性。具体的战斗设计、数值、阶段机制待后续细化，但设计原则是：

- 每个Boss的不同阶段有利于不同操作风格，但不彻底封死任何流派
- 后续Boss可设计更强的克制关系，逼迫玩家发展副线

## 七、叙事暗线

暗线不需要在游戏里明说，散落在环境叙事中——废墟里的壁画、残留的信件、锻造台旁的铭文、Boss击败后闪过的画面碎片。想挖的玩家能拼出来，不想挖的照样打得爽。

### 7.1 三层真相

**第一层（表面叙事）**：骑士团堕入异端，被王国正义审判。主角来清剿余孽。

**第二层（核心真相）**：骑士团没有堕落。他们发现了异力的真正来源在王都内部，试图揭露，被以"异端"之名灭口。大审判是政治清洗，不是正义制裁。

**第三层（深层真相）**：主角本身就是王国计划的一部分——派出经过改造的"审判者"去收割骑士团身上积累了几十年的异力，带回来给王国使用。审判者不是执行正义的人，是收割工具。用完之后大概率也会被以同样的罪名清除。

第三层可留给续作或DLC，不需要在第一版里完全揭示。

### 7.2 环境叙事示例

- 荒野区域散落的骑士日记碎片，记录着边境日常和对王国的忠诚
- 防御工事的墙壁上刻着骑士团的信条和战友名字
- 教堂壁画描绘骑士团的荣耀历史，但有人用利器划掉了王国的纹章
- 锻造台旁的铭文记录了异力稳定技术的研发过程
- 某个房间角落有一封没寄出的信，收信人是王都的某个人，内容暗示骑士团曾试图通报异力来源的发现

## 八、关卡结构与世界观对应

| 关卡元素 | 世界观叙事 |
|---------|----------|
| 战斗房间 | 骑士团旧堡垒的不同区域——林地、城墙、营房、训练场、教堂 |
| 结晶点 | 骑士团的异力稳定装置，战斗后用于凝固获取的异力 |
| 祭坛（天赋兑换） | 骑士团的锻造台，用凝固的异力改造自身 |
| 商店 | 边境补给站的残余物资，或还保持理智的骑士在做交易 |
| 事件 | 废墟中的发现——一封信、一段壁画、一个濒死的骑士 |
| 属性强化房 | 找到了骑士团留下的装备或训练设施 |
| Boss房 | 某个重要骑士的最后据点 |

## 九、待设计项

1. **Boss具体设计**：每个Boss的完整战斗机制、阶段切换、对话内容、数值
2. **精英怪细化**：异铸骑士的具体分类（剑/盾/弓等）、数值与行为
3. **最终Boss设计**：打完骑士团团长之后的最终对抗——可能是王国派来的更高级审判者，也可能是异力源头本身
4. **区域视觉设计**：三个区域的像素美术风格参考与色调规划
5. **环境叙事具体内容**：每个区域的信件、壁画、铭文等叙事道具的完整文案
6. **叙事结局设计**：主角发现真相后的选择——回去揭露王国？继续深入寻找异力源头？还是其他可能？
